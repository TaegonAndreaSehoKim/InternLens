# InternLens

InternLens is a personalized internship application strategy optimizer that ranks job postings by candidate fit, explains ranking signals, and recommends apply-now / apply-later / skip actions.

## Project Goal
This project helps a graduate student prioritize internship applications by combining profile analysis, semantic retrieval, ranking, and decision support.

## Planned Features
- Candidate profile analysis
- Internship retrieval and ranking
- Explainable fit scores
- Apply-now / apply-later / skip recommendations
- Feedback-based reranking

Project setup in progress.
